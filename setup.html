إنشاء مستودع GitHub

  1. اذهب إلى GitHub - New Repository
  2. أدخل اسم المستودع: forex-cafe
  3. اختر Private أو Public حسب رغبتك
  4. اضغط "Create repository"

ثم قم بتهيئة Git ورفع الملفات:

# Initialize git git init # Add all files git add . # Commit git commit -m "Initial commit - Forex Cafe" # Add remote (استبدل USERNAME باسم المستخدم) git remote add origin https://github.com/USERNAME/forex-cafe.git # Push git branch -M main git push -u origin main
2

إنشاء قاعدة بيانات Neon

  1. اذهب إلى neon.tech وأنشئ حساب مجاني
  2. اضغط "Create a project"
  3. أدخل اسم المشروع: forex-cafe
  4. اختر أقرب منطقة (مثل: AWS EU Frankfurt)
  5. اضغط "Create project"
بعد إنشاء المشروع، انسخ Connection string من Dashboard.
سيكون بهذا الشكل: postgresql://user:pass@host/db?sslmode=require

بذر البيانات الأولية:

أنشئ ملف .env في المشروع وأضف:

DATABASE_URL=postgresql://user:password@host/database?sslmode=require JWT_SECRET=your-super-secret-key-change-this-in-production

ثم شغّل أمر البذر:

npm install npm run db:seed
3

النشر على Vercel

  1. اذهب إلى vercel.com وسجّل بحساب GitHub
  2. اضغط "Add New Project"
  3. اختر مستودع forex-cafe
  4. في قسم "Environment Variables"، أضف:
DATABASE_URL = postgresql://user:password@host/database?sslmode=require JWT_SECRET = your-super-secret-key-change-this
  1. اضغط "Deploy"
تهانينا! بعد النشر، ستحصل على رابط مثل:
https://forex-cafe.vercel.app
4

إنشاء حساب المدير

  1. افتح الموقع بعد النشر: https://your-site.vercel.app/login.html
  2. اضغط "حساب جديد"
  3. أنشئ أول حساب - سيكون تلقائياً حساب Admin
  4. سيتم توجيهك إلى لوحة التحكم
ملاحظة: أول مستخدم يسجّل في الموقع يحصل تلقائياً على صلاحيات المدير (Admin). جميع المستخدمين بعده سيكونون "مشتركين" عاديين.
5

ربط دومين مخصص (اختياري)

  1. في Vercel، اذهب إلى Settings > Domains
  2. أضف دومينك: forex.cafe
  3. أضف سجلات DNS في مزود الدومين:
Type: A Name: @ Value: 76.76.21.21 Type: CNAME Name: www Value: cname.vercel-dns.com
